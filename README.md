# Python Full Stack Personal Portfolio

## Overview
This is a personal portfolio and blog application built using **Python (Flask)** and **SQLite**. It features a modern, responsive design with a clean Sky Blue and Gray color scheme. The project demonstrates full-stack development skills including backend API development, database management, and frontend UI implementation.

## Features
-   **Portfolio Showcase**: Dynamic project section with detailed views.
-   **Blog System**: Read and manage blog posts.
-   **Admin Dashboard**: secure interface to manage content and view contact messages.
-   **Contact Form**: Functional form that saves messages to the database.
-   **Responsive Design**: Works perfectly on mobile and desktop.

## Tech Stack
-   **Backend**: Python, Flask, SQLAlchemy
-   **Database**: SQLite
-   **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
-   **Deployment**: Ready for PythonAnywhere or similar platforms.

## How to Run Locally
1.  Clone the repository:
    ```bash
    git clone https://github.com/manjunathreddy25/personal-portfolio.git
    cd personal-portfolio
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the application:
    ```bash
    python app.py
    ```
4.  Open `http://127.0.0.1:5000` in your browser.
